MY COMMENTS.

My Comments.✍

Reflection.

Adventure and self-reflection are the main concept around the whole design of this website. On the home page, people first encounter the image of a blacked-out road sign. This symbolise the mystery, the uncertainty of where the website is going to guide you. At the same time, it is fun, exciting which demonstrated by the vibrant background colour behind the black road sign. Following the next three webpages, the audience can see three different open-road images which all lead to different scenery and destination and this destination is information about me. Again, this ties up with the theme – adventure.
From the past page, it guides the audience to a cold, washed-out place symbolises past, my past. In contrast to the future, it leads the audience to a bright, warm scenery which symbolise my future.
At the bottom of each page has a quote for the audience to take home with. This hopefully provides the audience some self-reflection and each quote correlates to the context of each page.

Technical.

Structure

The website has four html webpages, home, past, future and comments and one CSS stylesheet. All 4 webpages follow the same structure as the page divided into two sections. On the top section, menu bar and introduction text can be seen. Then the bottom section where the main text content and the footer located. The navigation use hyperlink to navigate to each individual page on this website. The menu bar are on the top of the page provide easy navigation for the reader.

1. HTML structure

Head

Inside the <head> element, two main <meta> tags are included. Firstly, the Unicode character set UTF-8. This Unicode standard covers all the character, punctuation, and symbol in the webpage. Next viewport meta tag. This is to ensure that the website can be view and fit to any device screen size. Website icon is shown by using the <link> tag, this also used to link the CSS stylesheet for the website. The use of <title> tag provide a brief description of the page on the browser tab.

Section

<section> tag divided the webpage into section as shown in Figure 2, this is to separate the content from the other information such as navigation. The use of <nav> tag provides the navigation and menu bar on the website; this also includes the logo of the website. <li> element is used to organise the menu bar.

Content

The text content in the second section is placed inside another <section> tag to separate the image and the text. <p> tags are used to divided into different paragraphs. The use of <br> to break a sentence into a new line as shown in the next line.
The use of <caption> and <figcaption> to pair with the image itself as it provides another text description to the images.

Footer

The footer is divided using <footer> element, and in the footer element, it contains information about the copyright and image source of each page. A back-to-top button which can navigate the reader back to the top of the webpage.

2. CSS structure

CSS is responsible for the syling and position of each and single one of the element, class, id in the website. All content are placed inside a border-box which allows the padding and border included in an element's total width and height. Here are the summary of some CSS selector used in this website.

Aesthetics.

Colour

The use of pastel colour such as antique white as a background colour on the text section, the reason being that it has high brightness and low saturation meaning that they contain less grey, glooming colour which provides a calming and peaceful vibe to the website. The use of Indian red on the quote recapture the eye of the audience as they might lose focus after reading a long text and provide them a change of colour and scenery to the eye.

Font

Two main font family are used on the website. First, Ariel, this provides an easy-to-read text unlike font like cursive, and it is one of the most common font styles using across the world, this gives the reader a sense of familiarity. Second, the use of Courier New font family on the quote provides a change to separate the main text to the quote.

Navigation

The navigation has an opaque colour background, this is to contrast the background and the foreground of the page. Also, when a mouse point hovers over, a box highlight shows up, this is to provide a sense of direction and links to follow.

Back-to-top

The back to top button located on the footer has been set using the function in CSS: smooth scrolling, this is to prevent disorientation to the audience, and it smoothly scrolls to the top, it gives the audience the scene of where they are in the page. The link colour is left with standard blue hyperlink colour to inform the audience that this button is clickable. Try go to the bottom and pay attention to the smooth scrolling.

Others

The use of icon and logo gives identity to the website. The icon design using the website www.favicon.cc. And the logo has a hidden message - zoom into the logo! and the code can be decipher using rot13, this reinforces one of the main concepts of the website – adventure/mystery.
The website structure and position stay constant and centred as it provides consistency and continuity and directs the eye of the audience and make them focus on the centre.
The use of white space next to the text allows the audience focus on what is the most important. All hyperlink on this website opens into a new browser tab to avoid confusion, this is done by using <target> attribute inside the anchor tag.

Accessibility.

The website is built with the idea that it is easily accessible for people with disability such as colour blindness, visual impairment , cognitive/motor neurone disorder by following the guideline standard for web accessibility from W3C and WebAIM.

Colour blindness

The site has been tested using third party colourblind filter to make sure the colour of the image and text are easily readable for all types of colour blindness. This includes Protanopia – red colour blindness, Deuteranopia – green colour blindness, Tritanopia – blue/yellow colour blindness and Greyscale/achromatopsia – total colour blindness. The use of dark colour text on light colour background provides a great contrast to the reader e.g., black on white. No two similar colour are overlapping on the website e.g., red on dark red.

Visual impairment – total blindness

The use of <alt> tag on an image provide text alternative for non-text content. In this way, for people using reader mode on a website can listen a description of an image. The page links are all accessible using tab key on keyboard for easy page navigation. The site is robust, it is still functional without the use of CSS.


Cognitive/motor neuron disorder

Similar to the use of tab key and up and down arrows. The site can be easily assessable with just a tap of a single key for people that are using assistive technology such as a mouth stick or head wand; this is a solution for people who cannot use a mouse.

Other aspects

As technology grows, people are using different device to view a website, such as phone, tablet, or laptop. The site is responsive to all device screen size to make sure the content fit for every device. This is archived by using @media CSS properties to adjust the content based on the device width and height and also the orientation. When left clicking a website on Google chrome, the type of phone device or tablet device and the orientation can be set and view. Please try resize the window or go to inspect on Google Chrome and switch to any device to observe how responsive the text, menu and image is!!

When zooming in and out of the website the text will automatically adjust to the appreciate size. This is achieved by also using @media CSS properties and also using non-fixing size value such as px, instead using vw unit meaning viewport width, this way the text size will follow the size of the browser window. When adjusting the windows size, image will change according to the screen size. This is done using <source> attribute, different picture set base on the window size.
